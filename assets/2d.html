Overview

2D Assets

비교적 작은 용처에서 오늘의집 주요 서비스의 성격을 시각적으로 대표하고, 사용자의 서비스 선택과 사용을 직관적으로 안내하는 평면 그래픽 자산입니다.

Form Principles

2D 자산만의 결정.

2D 자산을 단순한 일러스트가 아닌 시스템의 일부로 만드는 네 가지 결정.

3D와 같은 모티브, 평면의 어법

3D 자산과 같은 사물을 다루지만, 표현은 평면화 — 입체의 결정 없이 형태와 색만으로 같은 인상을 만듭니다.

단순한 형태, 깊이 있는 색

형태는 가장 단순한 매스로 정리하고, 깊이감은 색의 단계로 만들어냅니다. 그라디언트와 분리된 면이 평면 위에서 입체감을 대신합니다.

작아도 무너지지 않는 단순함

2D 자산은 작은 크기에서도 사용됩니다. 축소되었을 때 형태가 무너지지 않도록, 처음 디자인할 때부터 가장 작은 사이즈를 함께 검토합니다.

공통 색 시스템 위의 변형

색의 변형은 자유롭지 않습니다. 라이브러리 전체가 공유하는 색 시스템 안에서 결정되어, 한 화면 안에서 다른 자산과 같은 톤으로 읽힙니다.

Workflow

제작 방식.

01

모티브 정렬

3D 자산과 공유할 모티브를 먼저 정합니다. 새 모티브일 경우 3D·2D 양쪽에서 동시에 검토합니다.

02

형태 정리

공통 그리드 위에서 가장 단순한 매스로 형태를 정리합니다. 디테일은 색의 단계로 옮겨갑니다.

03

색 적용

공통 색 시스템 안에서 자산의 결을 잡습니다. 그라디언트와 면 분리는 입체감의 단서로 작동합니다.

04

사용처 변형

주요 사용처에 맞춰 변형을 함께 정리해 공급합니다. 한 모티브가 여러 사용처를 일관되게 책임집니다.
